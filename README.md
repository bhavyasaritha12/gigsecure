# Micro Insurance Portal for Gig Workers

A digital platform designed to provide affordable and accessible micro-insurance services for gig workers such as delivery partners, drivers, freelancers, and daily wage earners.
## Problem Statement
Gig workers often work without financial security or insurance coverage. Existing insurance systems are expensive, complex, and not designed for workers with unstable income. This project helps bridge that gap by offering accessible micro-insurance services through a digital platform.

## Objectives
- Provide low-cost insurance plans for gig workers
- Simplify policy registration and claim processes
- Increase financial security for workers in the gig economy
- Offer mobile-friendly and easy-to-use services
- Create awareness about insurance benefits

## Key Features
- User Registration & Login
- Insurance Plan Recommendations
- Affordable Premium Options
- Digital Claim Submission
- Emergency Support Assistance
- Secure Payment Integration
- User Dashboard for Policy Tracking
- Mobile Responsive Design

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Python / Firebase
- Database: MongoDB / MySQL
- Tools: GitHub, VS Code
